API :: About

Welcome

Welcome to the FindApp API. This API serves the FindApp ecosystem.

This API allows community use of the FindAppCo database listings sorted by their signature tags.

Developers will be able to retrieve all listings and their community-generated tags for project use.

NOTE : current API listings are wellness industry pages available from our WellFind listing app. The FindApp ecosytem will soon expand to allow for business listings of every variety, giving the community the ability to truly synchronize their values, needs, and patronage.

Docs

API: Retrieve All Current Listings

GET:

https://wellfindapi.herokuapp.com/api/listings/read.php

See Sample Output

See the Tag Search tab to use this API with a filter query

API: Retrieve Listings By Tag Search

GET:

https://wellfindapi.herokuapp.com/api/listings/search.php?search=%TAG%


Query Params
(please include %-%):

search | %TAG%
